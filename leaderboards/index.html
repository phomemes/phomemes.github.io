Participate in the PhoMemes Data Challenges.

The goal of these leaderboards are to encourage "coopetition" (cooperation + competition) among groups working on images in social media, as they pertain to disinformation, misinformation, hate speech, and propagation.

Systems participating in these challenges will be given data as described here.

To participate, simply follow the directions in our GitHub repo and issue a pull request.
